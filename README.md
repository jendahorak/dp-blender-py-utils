# dp-blender-py-utils

Utility scripts for manipulating City Engine 3D models exports of Brno buildings in Blender.

## Scripts

- clean-models.py
- clean-up.py
- remap-colors.py
- reset-project-data.py
- vertex-colors.py

## Requirements

These scripts require Blender 2.8 or later.
